# 文件共享系统 (File Share System)

这是一个轻量级的局域网文件共享工具，旨在简化同一 WiFi 或热点下不同设备（Windows, Mac, Android, iOS）之间的文件传输。

## 主要功能

*   **跨平台支持**：只要有浏览器即可访问，无需在客机上安装任何软件。
*   **文件上传/下载**：支持从主机下载文件，或从任意设备上传文件到主机。
*   **拖拽上传**：桌面端支持将文件直接拖入网页进行上传。
*   **移动端适配**：针对手机端优化的上传按钮和界面布局。
*   **实时进度**：显示上传进度条和实时传输速度。
*   **自动刷新**：文件列表会自动轮询更新，无需手动刷新页面。
*   **文件管理**：支持直接在网页上删除已共享的文件。

## 技术栈

*   **后端**：Python (Flask)
*   **前端**：HTML5, CSS3, JavaScript (原生)
*   **依赖管理**：pip

## 使用方法

### 1. 环境准备
确保你的电脑（作为主机）已安装 **Python 3.x**。

### 2. 启动服务
双击运行项目根目录下的 **`run.bat`** 脚本。
*   首次运行时，脚本会自动安装所需的依赖库（Flask）。
*   启动成功后，控制台会显示本机的局域网 IP 地址和端口号（默认为 5000）。

### 3. 访问共享
在其他设备（如手机、iPad 或另一台电脑）的浏览器中，输入控制台显示的地址。
*   例如：`http://192.168.1.5:5000`
*   **注意**：请确保所有设备连接在同一个 WiFi 网络下，或者连接到了主机开启的热点。

### 4. 常见问题
*   **无法访问**：请检查主机的 Windows 防火墙设置，尝试允许 `python` 通过防火墙，**建议仅在专用网络中使用本代码**。
*   **上传失败**：请确保共享文件夹 `shared_files` 没有被其他程序占用，且磁盘空间充足。

## 目录结构

*   `app.py`: Flask 后端核心逻辑
*   `templates/index.html`: 前端界面代码
*   `run.bat`: Windows 启动脚本
*   `requirements.txt`: Python 依赖列表
*   `shared_files/`: 默认的文件存储目录

## 注意事项
不建议使用本软件传输隐私文件或者其他重要文件，不对传输过程中的安全作任何保证。
